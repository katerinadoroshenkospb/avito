### Для запуска АТ:
1. Склонировать репозиторий:
`git clone https://github.com/katerinadoroshenkospb/Avito`
2. Перейти в склонированный репозиторий, в подпапку с файлом gradlew.bat
3. Для запуска АТ использовать команду
`gradlew test`
4. Отчёт о прогоне АТ будет сфоримрован. Для его просмотра открыть файл build/reports/tests/test/index.html в браузере. 

### Структура фреймворка:
```
├── src/
    ├── main/
    └── test/
        └── constants                   # Константы проекта
        └── models                      # POJO
        └── providders                  # содержит методы для отправки запросов
        └── services                    # различные сервисные классы, используемые в АТ
        └── tests                       # разработанные АТ
        └── utils                       # утилиты для создания тестовых данных
├── build.gradle                        # Конфигурация сборки, добавлен модуль common
├── README.md                           # Документация
└── TESTCASES.md                        # Разработанные тест-кейсы, используемые для создания АТ

├── task1                               # Папка 1 задания
    ├── img                             # папка со скриншотами багов
    ├── BUGS                            # файл с описанием найденных багов
```
